# springwebapp-catalog-staff
Это тестовый проект Spring RESTful Web Service.

#### Задание 

>В наличии SQL база данных (любая), в которой необходимо хранить список сотрудников (Фамилия, Имя, Отчество, Должность, Дата рождения). Необходимо написать веб-приложение, реализующее функциональность поиска сотрудников по их данным, а также отображение таблицы с результатами поиска. Серверная часть приложения должна быть реализована на языке Java (рекомендуется посмотреть на Spring, в частности Spring-bootstrap), при реализации клиентской части рекомендуется использовать AngularJS.

#### Требования к запуску
* Maven
* PostgreSQL 9.4
* IDE (желательно IntelliJ IDEA 2016)

#### Запуск
После открытия проекта соответствующей средой разработки, необходимо указать ваши настройки БД в файле `application.properties`. Данный проект автоматически создаст необходимую ему базу в схеме по умолчанию (`public`) и заполнит ее тестовыми данными, после запуска проекта. Запуск проекта осуществляется с класса `Application` пакета `ru.devjava.averchenkov.staff`.

#### Примечание
Надо заметить, что данное проектное решение не является готовым решением, а только неким прототипом который может послужить разработке реально работающего приложение. Так например, количество обращений на сервер стоит сократить, делая один запрос на 100 сотрудников и формируя страницы в AngularJS. В архитектурном плане StaffService стоит разбить на еще один класс, который был реализовывал общие для всех сервисов доступа к данным функции. Таким образом, данная разработка имеет смысл жить, только как демонстрационное решение.
